表格处理

表格样式

  • table-layout

    • auto 默认，列之间会自动伸缩
    • fixed
      • 当配合设置表格宽度时，固定每个列单元格大小相同

  • border-collapse：边框塌陷模式

    • separate 分离模式
    • collapse 塌陷模式，边框会产生合并

  • border-spacing：单元格间隙，只适用于分离模式

衍生样式

斑马条纹

  • tr:nth-child(odd) : 奇数
  • tr:nth-child(even) : 偶数
