背景处理

默认背景显示的区域

盒模型的内边距和内容区，padding + content

背景色

  • background-color

背景图片

背景图片可以和背景色一起用，图片透明区域和图片没有完全填充的区域会显示背景色

  • background-image 资源

    值 描述
    none 不指定 默认
    url 图片资源路径

  • background-repeat 平铺

    值 描述
    repeat 双向平铺 默认
    no-repeat 不重复
    repeat-x 水平平铺
    repeat-y 垂直平铺

  • background-size 大小

    值 描述
    auto 保持图片尺寸 默认
    contain 保持图片尺寸缩放，完全包裹图片
    cover 保持图片尺寸缩放，图片完全填充背景
    单个值 同时设置图片的宽度和高度
    两个值 依次设置图片的宽度和高度，其中一个值设置为 auto，图片会缩放

  • background-position 背景定位

    值 描述
    center 居中
    top 或 bottom 上下对齐
    left 或 right 左右对齐
    百分值 默认情况下根据元素滚动框和图片大小的差值计算
    像素 相对背景区域的偏移

背景固定

background-attachment

值 描述 定位影响
scroll 不跟随所在元素的滚动框滚动 定位使用的百分值会基于滚动框大小来计算
fixed 不跟随页面滚动 类似固定定位，定位使用的百分值会基于视口大小来计算
local 跟随所在元素的滚动框滚动 定位使用的百分值会基于元素内容大小来计算

背景区域 ie9

background-origin

  • border-box
    背景图片的摆放以 border 区域为参考
  • padding-box
    背景图片的摆放以 padding 区域为参考
  • content-box
    背景图片的摆放以 content 区域为参考

剪裁 ie9

background-clip

  • border-box
    背景延伸至边框外沿（但是在边框下层）。
  • padding-box
    背景延伸至内边距外沿。不会绘制到边框处。
  • content-box
    背景被裁剪至内容区外沿。
  • text （实验功能，不建议使用）
    背景被裁剪成文字的前景色。
