转化为数字

把其他类型的数据转化为数字常用以下方式：

  • 使用 Number 函数
  • 使用 parseInt 函数
  • 使用 parseFloat 函数
  • 使用 + 一元运算符
  • 使用 - 一元运算符

+ 和 - 运算符功能强大，可以用于计算、正负值或类型转换。

类型转换小结

小结：对于复杂的隐式类型转换，应该进行类型判断，不应该直接计算。下一节我会介绍判断数据的类型

类型判断

常用来进行类型判断的方式有两种，一个是通过运算符 typeof，或者通过一个内置函数：

  • typeof 关键字
  • Object.prototype.toString.call 方法

也还要其他判断类型的方式，在此不做过多讨论。

typeof

typeof 的使用很简单，它只有一个操作数：

使用 typeof 关键字描述一个数据会返回这个数据的类型，是一个字符串，一般通过逻辑分支来来判断这个数据的类型，比如：

使用 typeof 会返回以下字符串：

类型 typeof 返回的字符串
Null 空值 object
Undefined 未定义 undefined
String 字符 string
Number 数字 number
Boolean 布尔 boolean
对象 object
数组 object
函数 function

注意，使用 typeof 判断数组和 null 的时候，返回的是 object 和对象一样。其中 typeof null 是 JS 程序中一个悠久的 bug，这个 bug 不会修复，因为现在很多 JS 代码中依赖这个 bug 开发代码。

也可以使用一个内置函数来判断数据类型：
