你还可以在交互式控制台实时编写 JavaScript 代码，【f12】打开控制台，然后选中 Console 模块，键入 alert(123)，代码依旧会执行。

基本语法

  • JS 代码通过 ; 标记一个语句的结束，; 是可以省略的。
  • 多余的留白和缩进对代码没有影响。
  • JS 代码区分大小写，不像 HTML 一样。

注释

JS 中有两种注释方式：

  • 单行注释：使用 // 来注释单行文本，单行注释仅作用双反斜杠后面的内容
  • 多行注释：使用 /**/ 来注释多行文本
