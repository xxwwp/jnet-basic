## 无语义元素

HTML 中有两个完全没有语义的元素如下：

| 元素   | 类别 | 语义 | 属性 | 备注                  |
| ------ | ---- | ---- | ---- | --------------------- |
| `div`  | 块级 | 无   |      | 划分任意 **块级区域** |
| `span` | 内联 | 无   |      | 划分任意 **行内区域** |

`div` 和 `span` 虽然没有语义和样式，却是使用最频繁的元素。如果你无法找到一个确切的元素来包裹内容，那么就可以使用 `div` 和 `span` 元素代替，比如卡片，广告，提醒，图标等等。

示例：

```html
<!-- 使用 div 和 span 划分一张卡片的各个部分 -->
<div>
  <span>卡片标题</span>
  <p>卡片的描述...</p>
  <p>
    <span>类型：明星片</span>
    <span>时间：xxxx-xx-xx</span>
  </p>
</div>
```

HTML 并没有提供卡片相关的元素，我们就可以 `div` 或者 `span` 元素来划分卡片的内容。
