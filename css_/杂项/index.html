opacity

opacity 属性用来改变元素的透明度，包括它的子元素。opacity 的值是数字 0~1，0 为完全透明，1 为完全显示。

隐藏补充

分析一下三条声明的不同：

声明 结果 影响 交互
display:none 元素没有盒模型，不会参与布局 不渲染 + 不保留布局 不参与交互
visibility:hidden 元素不可见 不渲染 + 保留布局 不参与交互
opacity:0 元素完全透明 渲染 + 保留布局 参与交互，如果是表单或者超链接等等，依旧可以点击
