文本处理

字体样式

  • 字体库：font-family

    • 本地字体

    • 网络字体

      win10 的字体在 C:\Windows\Fonts

  • 大小：font-size

  • 颜色：color

  • rgb 三原色

  • 粗细：font-weight

  • 装饰线：text-decoration

    • none 无
    • underline 下划线
    • overline 上划线
    • line-through 中划线

  • 斜体

    • font-style:normal
    • font-style:italic

  • 字符间距：letter-spacing（作用于字母或汉字）

  • 单词间距：word-spacing（只可以作用于单词，汉字按字符计算，汉字没有单词，所以此项对汉字无效）

列表

  • list-style-type ：设置用于列表的项目符号的类型，例如无序列表的方形或圆形项目符号，或有序列表的数字，字母或罗马数字。

    值 描述
    none 无标记。
    disc 默认。标记是实心圆。
    circle 标记是空心圆。
    square 标记是实心方块。
    decimal 标记是数字。
    … …

    其他参见表

  • list-style-position ：设置在每个项目开始之前，项目符号是出现在列表项内，还是出现在其外。

    值 描述
    outside 标记盒在主块盒的外面。
    inside 标记盒是主要块盒中的第一个行内盒，处于元素的内容流之后。

  • list-style-image ：允许您为项目符号使用自定义图片，而不是简单的方形或圆形。

  • list-style 简写

    • Formal syntax: <'list-style-type'> || <'list-style-position'> || <'list-style-image'>

链接

  • 链接的伪类：已经介绍过，不再介绍，直接使用

    伪类 选择的范围
    :link 没有访问过
    :visited 已经访问过
    :active 激活
    :hover 悬停
    :focus 聚焦
