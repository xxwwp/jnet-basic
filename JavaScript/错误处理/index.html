错误处理

异常

如果程序不能正确的运行，那么就会抛出一个异常，异常可能由语法错误引起，也可能是运行时错误的计算引起。于是错误被分为以下两类：

  • 语法错误
  • 运行错误

name 错误的名称，Error 函数生成的对象名称为 Error
message 错误的消息，就是传入 Error 函数的字符串
stack 错误生成时，当前程序的调用堆栈
