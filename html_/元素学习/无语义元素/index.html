无语义元素 div 和 span

HTML 中有两个完全没有语义的元素如下：

元素 类别 语义 属性 备注
div 块级 划分任意块级区域
span 内联 划分任意行内区域

这两个元素没有任何样式和语义，一般用来划分页面，比如：

HTML 并没有提供卡片相关的元素，我们就可以 div 或者 span 元素来划分卡片的内容。

div 和 span 虽然没有语义和样式，却是使用最频繁的元素，因为它们被用来标记所有没有相关语义的内容。比如卡片，广告，折叠框，图标等等。
